
#本平台使用了前端[Bootstrap](http://www.bootcss.com/)框架和后端[Flask](http://flask.pocoo.org/)框架
#数据库使用了[SQlite](https://www.sqlite.org/docs.html)

#安装环境
### Requirements
    pip install flask
    pip install SQLAlchemy
    
#文件结构
1. app.py --- 主程序
2. data.sqlite  --- 数据库，保存用户注册信息
3. static --- 静态路径，
   1. bad_result 存放视频检测处理结果
   2. css --- 存放平台要使用的css文件
   3. img --- 存放平台使用的图片
   4. js --- 存放平台要使用的js文件
   5. videos --- 存放用户要进行检测的视频数据
4. templates --- 存放前端html文件
5. data --- 用来备份数据的文件夹

#程序运行
在终端下运行　app.py　程序
python app.py
